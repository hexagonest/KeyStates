Welcome to KeyStates

Pretty straightforward little script. Just makes checking for keydowns easier. Useful for games.

<script src="js/keystates.js"></script> <script> Keys.init( document ); //initialise key monitoring for the document element console.log( Keys.isDown( 65 ) ); //alert to check if the key a (keycode 65) is being held down </script>
